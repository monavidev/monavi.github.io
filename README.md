# Monavi GitHub Pages landing page

Static landing page for Monavi's React Native npm packages.

- GitHub: https://github.com/monavidev
- npm: https://www.npmjs.com/~monavi
- Contact: monavidev@hotmail.com

## Files

```txt
monavidev.github.io/
├── index.html
├── styles.css
├── script.js
├── 404.html
└── assets/
    └── monavi-mark.svg
```

## Deploy

Create or use the GitHub repository named:

```txt
monavidev.github.io
```

Then push these files to the `main` branch.

```bash
git add .
git commit -m "Create Monavi landing page"
git push
```

GitHub Pages URL:

```txt
https://monavidev.github.io/
```
