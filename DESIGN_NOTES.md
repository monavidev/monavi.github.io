# Monavi landing page design notes

## Direction

Premium React Native package studio. The design is intentionally restrained: large type, strong spacing, few colors, and clear product hierarchy.

## Typography

- Sans: Geist
- Mono: Geist Mono
- Hero headline: very large, tight line-height, negative tracking
- Body text: 16–22px, readable line-height

## Color palette

- Canvas: warm off-white `#f7f3ec`
- Ink: near-black `#12120f`
- Muted text: `#756f64`
- Accent: cobalt blue `#1947e5`

## Composition

- Hero uses a 52/48 split on desktop.
- Left side communicates the brand and value.
- Right side gives proof: install commands and real package links.
- Package section uses one featured card plus two supporting cards.

## Hierarchy

1. Brand and promise
2. npm / GitHub call to action
3. Package proof
4. Package details
5. Principles
6. Contact
